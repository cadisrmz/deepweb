Deepweb proxy

Deepweb is a tool for easy browsing deep web. Currently we provide a Proxy auto-config for this purpose, on my linode, providing a deep web gateway.

How to use

Set your browser proxy to PAC URL: http://liruqi.github.io/deepweb/dw.pac

Security

  • Currently we only proxy for .onion domains.
  • We do not record anything, but the link from end user to my gateway is un-secure. use https.
  • Get what you need to know. Do not trust anyone in deep web.
